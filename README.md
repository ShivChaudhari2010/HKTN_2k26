# 🚀 AIFlow — Next-Generation AI Automation Platform

A premium, responsive SaaS landing page developed for **FrontEnd Battle 3.0 – VibeCoding Competition** hosted on **Unstop**.

The project demonstrates modern frontend engineering, responsive UI architecture, accessibility, performance optimization, and state management using React.

> Competition Date: **26 June 2026**

---

# 📖 Project Overview

AIFlow is a fictional AI-powered workflow automation platform designed to showcase modern frontend development practices.

The landing page focuses on creating a premium SaaS experience with smooth interactions, semantic HTML, responsive layouts, optimized rendering, and production-ready component architecture.

The project was built by following the competition constraints, including custom-built UI components, CSS-only animations, and performance-focused rendering strategies.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

Go to the project folder

```bash
cd your-repository
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
